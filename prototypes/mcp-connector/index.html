MCP = Model Context Protocol

An open standard that lets AI assistants securely connect to business tools and data sources. Before MCP, AI only knew what you typed. With MCP, it can look things up.

Developed by Anthropic and adopted by OpenAI and Google ‚Äî it's the universal adapter for AI tools. Staircase's MCP connector uses this standard to give your AI access to customer intelligence.

Platform Setup

Connect in 5 Minutes

One-time setup, then just ask questions. Choose your AI tool below.

1

Open Claude Desktop Settings

Click Claude in the menu bar (macOS) or File (Windows) ‚Üí Settings ‚Üí Developer ‚Üí Edit Config

2

Add Staircase MCP Configuration

{
  "mcpServers": {
    "staircase-ai": {
      "command": "npx",
      "args": ["-y", "mcp-remote",
        "https://mcp.staircase.ai/sse"]
    }
  }
}
3

Save and Restart Claude Desktop

Save the file, then fully quit and relaunch Claude Desktop (just closing the window is not enough).

4

Authenticate

A browser window will open automatically. Sign in with your Google or Microsoft work account and grant permission.

5

Test the Connection

Ask Staircase: What topics are trending in customer conversations?
Config file location:
macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
Windows: %APPDATA%\Claude\claude_desktop_config.json
Subscription required: ChatGPT MCP requires a Pro, Team, or Enterprise subscription.
1

Open ChatGPT Settings

Click your profile icon (bottom left) ‚Üí Settings ‚Üí Connectors ‚Üí Advanced ‚Üí Toggle Developer Mode ON

2

Create Staircase Connector

Settings ‚Üí Connectors ‚Üí Create Connector (+)

Name: Staircase AI
URL: https://mcp.staircase.ai/mcp
3

Authenticate

Sign in with your Google or Microsoft work account when prompted. You should see a green "Connected" status.

4

Test the Connection

Use Staircase to find accounts with recent risk signals
Note: ChatGPT uses the /mcp endpoint, not /sse like Claude Desktop.
For developers: Gemini CLI requires a Google Cloud account with Gemini API access and Node.js installed.
1

Install Gemini CLI

npm install -g @google/gemini-cli
2

Create config file at ~/.gemini/config.json

{
  "mcpServers": {
    "staircase-ai": {
      "command": "npx",
      "args": ["-y", "mcp-remote",
        "https://mcp.staircase.ai/sse"]
    }
  }
}
3

Authenticate with Google

gcloud auth login
4

Authenticate with Staircase

On first query, a browser opens. Sign in with your work Google or Microsoft account and grant permission.

5

Test the Connection

Query Staircase for trending customer topics
